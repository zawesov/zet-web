Use the Makefile to compile, install, clean:
make -f Makefile, make -f Makefile install, make -f Makefile clean
mingw32-make -fMakefile.mgw, mingw32-make -fMakefile.mgw install, mingw32-make -fMakefile.mgw clean

Add libraries libevent, openssl.

By default, header files and lib file will be installed in
/usr/local/include/zetweb_ver/ and /usr/local/lib/libzetweb_ver.a
C:/MinGW/include/zetweb_ver/  and C:/MinGW/lib/libzetweb_ver.a

For visualStudio set "Character Set" to "Use Multi-Byte Character Set"

How to use the library see Documentation_ver.html, HowTo_ver.html, "./example".
